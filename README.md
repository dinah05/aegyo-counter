## ✨ 이런 앱이에요

엠키의 “호에엥…”, “우우우...”, “눈나~” 같은 애교를 들을 때마다  
`+` 버튼을 눌러 오늘의 애교 횟수를 기록합니다.

애교를 견딜수록 분노 게이지가 차오르고, 특정 횟수에 도달하면 업적을 획득할 수 있어요.

<br />

## 🔥 주요 기능

- ➕ 애교 횟수 증가 / 감소 / 초기화
- 😾 카운트에 따라 변화하는 분노 게이지
- 💬 랜덤 캐릭터 멘트
- 🏆 업적 달성 다이얼로그
- 🔊 버튼 클릭 효과음
- 📳 버튼 클릭 진동 피드백
- 📊 오늘 최고 / 이번 주 최고 / 전체 최고 기록 저장
- 💾 앱을 종료해도 기록 유지
